## 📱 概要

AVIVは、ホワイトノイズ再生と音楽プレイヤー機能を組み合わせたAndroidアプリケーションです。豊富なオーディオエフェクトとパーソナライズ設定を提供し、リラックス、集中力向上、睡眠の質の改善をサポートします。

---

## 📜 使用前の重要なお知らせ

初期使用時、アプリのログ機能はデフォルトで有効になっており、起動時のクラッシュログを記録します。パフォーマンスに若干の影響を与える可能性があります。設定で無効にすることができます。

このプロジェクトは現在、部分的なソースコードのみを公開しています。逆コンパイル、有料機能の注入、広告付きでの再配布は行わないでください。このプロジェクトは完全に無料です。正規版をサポートしてください。学習目的の場合は、作者に連絡して承認を得てください。

## ✨ 機能

### 🎵 音楽プレイヤー

- **マルチフォーマット対応**: MP3、WAV、FLAC、AACなど
- **3D空間オーディオ**: サウンドポジショニング、距離減衰
- **12バンドイコライザー**: プロフェッショナルイコライザー
- **クリエイティブエフェクト**:
  - Lo-Fi（ローファイ）
  - 8-bit（レトロゲームサウンド）
  - 水中効果
  - エイリアンシグナル
  - メガホン効果
  - ディストーション
- **空間リバーブ**: ルームサイズ、減衰時間、ウェット/ドライミックス
- **再生コントロール**: ピッチ調整（±12半音）、速度コントロール（0.3x - 3.0x）
- **プレイリスト管理**: プレイリストの作成と管理

### 🌊 ホワイトノイズ

- **複数のソース**: 様々な内蔵ホワイトノイズソース
- **オーディオエフェクト**: 各ソースごとに独立したエフェクト設定
- **スキャッターモード**: カスタマイズ可能な間隔でランダム再生
- **ボリュームコントロール**: 0-300%ゲインレンジ

### 🎨 パーソナライゼーション

- **カスタムテーマ**: HSV色カスタマイズ
- **タイマー**: 自動停止タイマー（最大23時間59分）
- **設定の永続化**: エフェクトパラメータの自動保存
- **ログ記録**: トラブルシューティング用のオプションログ

### 🎛️ ミキサーパネル

- **エフェクト強度**: 全エフェクトの0-100%調整
- **リアルタイムプレビュー**: 即座にエフェクトをプレビュー
- **クイックリセット**: ワンクリックでパラメータをリセット

## 📥 ダウンロード

### 要件

- Android 10.0 (API 29) 以上
- 少なくとも100MBの空きストレージ

### インストール

[Releases](https://github.com/byxixiaoshao/AVIV/releases)ページから最新のAPKをダウンロードしてください。

**手順**:
1. APKファイルをダウンロード
2. デバイスで「不明なソースからのインストール」を有効にする
3. APKファイルを開いてインストール
4. インストール後にアプリを起動

## 📖 使い方

### 音楽再生

1. トッププレイヤーバーを下にスワイプしてプレイヤーインターフェースを開く
2. ファイルピッカーで音楽ファイルを追加
3. ミキサーパネルでオーディオエフェクトを調整
4. プレイリストを作成・管理

### ホワイトノイズ

1. メイン画面でホワイトノイズソースを選択
2. ソースカードをタップして再生開始
3. カードを長押ししてエフェクトを設定
4. スキャッターモードでランダム再生

### オーディオエフェクト

- **イコライザー**: ミキサー → イコライザーパネル
- **クリエイティブエフェクト**: ミキサー → 音質パネル
- **空間リバーブ**: ミキサー → リバーブパネル
- **サウンドポジショニング**: ミキサー → 空間オーディオパネル
