## 📱 О приложении

AVIV - это Android-приложение, объединяющее воспроизведение белого шума и музыкальный плеер с богатыми аудиоэффектами и персонализированными настройками, помогающее пользователям расслабиться, улучшить концентрацию или качество сна.

---

## 📜 Важное уведомление перед использованием

По умолчанию функция ведения журнала включена при первом использовании для записи журналов сбоев при запуске. Это может незначительно повлиять на производительность. Вы можете отключить её в настройках.

Этот проект в настоящее время публикует только частичный исходный код. Пожалуйста, не декомпилируйте, не внедряйте платные функции и не распространяйте с рекламой. Этот проект полностью бесплатен - пожалуйста, поддерживайте официальную версию. Для учебных целей вы можете связаться с автором для получения разрешения.

## ✨ Функции

### 🎵 Музыкальный плеер

- **Поддержка множества форматов**: MP3, WAV, FLAC, AAC и другие
- **3D пространственное аудио**: Позиционирование звука, затухание расстояния
- **12-полосный эквалайзер**: Профессиональный эквалайзер
- **Креативные эффекты**:
  - Lo-Fi (низкая точность)
  - 8-bit (ретро игровой звук)
  - Подводный эффект
  - Инопланетный сигнал
  - Эффект мегафона
  - Искажение
- **Пространственная реверберация**: Размер комнаты, время затухания, микс влажный/сухой
- **Управление воспроизведением**: Регулировка высоты тона (±12 полутонов), контроль скорости (0.3x - 3.0x)
- **Управление плейлистами**: Создание и управление плейлистами

### 🌊 Белый шум

- **Множество источников**: Различные встроенные источники белого шума
- **Аудиоэффекты**: Независимая настройка эффектов для каждого источника
- **Режим разброса**: Случайное воспроизведение с настраиваемыми интервалами
- **Контроль громкости**: Диапазон усиления 0-300%

### 🎨 Персонализация

- **Пользовательские темы**: Настройка цвета HSV
- **Таймер**: Таймер автоматической остановки (до 23ч 59м)
- **Сохранение настроек**: Автосохранение параметров эффектов
- **Ведение журнала**: Опциональное ведение журнала для устранения неполадок

### 🎛️ Микшер-панель

- **Интенсивность эффектов**: Регулировка 0-100% для всех эффектов
- **Предпросмотр в реальном времени**: Мгновенный предпросмотр эффектов
- **Быстрый сброс**: Сброс параметров одним нажатием

## 📖 Использование

### Воспроизведение музыки

1. Смахните вниз верхнюю панель плеера, чтобы открыть интерфейс плеера
2. Добавьте музыкальные файлы через выбор файлов
3. Настройте аудиоэффекты с помощью микшер-панели
4. Создавайте и управляйте плейлистами

### Белый шум

1. Выберите источник белого шума на главном экране
2. Нажмите на карточку источника для начала воспроизведения
3. Долгое нажатие на карточку для настройки эффектов
4. Используйте режим разброса для случайного воспроизведения

### Аудиоэффекты

- **Эквалайзер**: Микшер → Панель эквалайзера
- **Креативные эффекты**: Микшер → Панель качества звука
- **Пространственная реверберация**: Микшер → Панель реверберации
- **Позиционирование звука**: Микшер → Панель пространственного аудио
