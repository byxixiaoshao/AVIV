## 📱 Sobre

AVIV é um aplicativo Android que combina reprodução de ruído branco com um player de música, oferecendo ricos efeitos de áudio e configurações personalizadas para ajudar os usuários a relaxar, melhorar a concentração ou a qualidade do sono.

## ✨ Recursos

### 🎵 Player de música

- **Suporte a múltiplos formatos**: MP3, WAV, FLAC, AAC e mais
- **Áudio espacial 3D**: Posicionamento de som, atenuação de distância
- **Equalizador de 12 bandas**: Equalizador profissional
- **Efeitos criativos**:
  - Lo-Fi (Baixa fidelidade)
  - 8-bit (Som de jogo retrô)
  - Efeito subaquático
  - Sinal alienígena
  - Efeito megafone
  - Distorção
- **Reverberação espacial**: Tamanho da sala, tempo de decaimento, mix molho/seco
- **Controle de reprodução**: Ajuste de tom (±12 semitons), Controle de velocidade (0.3x - 3.0x)
- **Gerenciamento de playlists**: Criar e gerenciar playlists

### 🌊 Ruído branco

- **Múltiplas fontes**: Várias fontes de ruído branco integradas
- **Efeitos de áudio**: Configuração independente de efeitos para cada fonte
- **Modo disperso**: Reprodução aleatória com intervalos personalizáveis
- **Controle de volume**: Faixa de ganho 0-300%

### 🎨 Personalização

- **Temas personalizados**: Personalização de cor HSV
- **Temporizador**: Temporizador de parada automática (até 23h 59m)
- **Persistência de configurações**: Salvamento automático de parâmetros de efeitos
- **Registro**: Registro opcional para solução de problemas

### 🎛️ Painel do mixer

- **Intensidade dos efeitos**: Ajuste de 0-100% para todos os efeitos
- **Visualização em tempo real**: Visualização instantânea dos efeitos
- **Redefinição rápida**: Redefinição de parâmetros com um clique

## 📥 Download

### Requisitos

- Android 10.0 (API 29) ou superior
- Pelo menos 100 MB de espaço de armazenamento livre

### Instalação

Baixe o APK mais recente da página [Releases](https://github.com/byxixiaoshao/AVIV/releases).

**Passos**:
1. Baixar arquivo APK
2. Ativar "Instalar de fontes desconhecidas" no seu dispositivo
3. Abrir arquivo APK para instalar
4. Iniciar o aplicativo após a instalação

## 📖 Uso

### Reprodução de música

1. Deslize para baixo a barra superior do player para abrir a interface do player
2. Adicione arquivos de música através do seletor de arquivos
3. Ajuste os efeitos de áudio usando o painel do mixer
4. Crie e gerencie playlists

### Ruído branco

1. Selecione uma fonte de ruído branco na tela principal
2. Toque no cartão da fonte para iniciar a reprodução
3. Pressione longamente o cartão para configurar efeitos
4. Use o modo disperso para reprodução aleatória

### Efeitos de áudio

- **Equalizador**: Mixer → Painel do equalizador
- **Efeitos criativos**: Mixer → Painel de qualidade de áudio
- **Reverberação espacial**: Mixer → Painel de reverberação
- **Posicionamento de som**: Mixer → Painel de áudio espacial
