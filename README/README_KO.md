## 📱 소개

AVIV는 백색 소음 재생과 음악 플레이어 기능을 결합한 Android 애플리케이션입니다. 풍부한 오디오 효과와 개인화 설정을 제공하여 사용자가 편안하게 휴식하고, 집중력을 높이며, 수면의 질을 개선할 수 있도록 도와줍니다.

---

## 📜 사용 전 중요 공지

초기 사용 시 앱의 로그 기능이 기본적으로 활성화되어 있어 시작 시 충돌 로그를 기록합니다. 성능에 약간의 영향을 줄 수 있습니다. 설정에서 비활성화할 수 있습니다.

## ✨ 기능

### 🎵 음악 플레이어

- **다중 포맷 지원**: MP3, WAV, FLAC, AAC 등
- **3D 공간 오디오**: 사운드 포지셔닝, 거리 감쇠
- **12밴드 이퀄라이저**: 전문 이퀄라이저
- **크리에이티브 이펙트**:
  - Lo-Fi (저품질)
  - 8-bit (레트로 게임 사운드)
  - 수중 효과
  - 외계 신호
  - 확성기 효과
  - 디스토션
- **공간 리버브**: 룸 크기, 감쇠 시간, 웻/드라이 믹스
- **재생 제어**: 피치 조정 (±12 반음), 속도 제어 (0.3x - 3.0x)
- **플레이리스트 관리**: 플레이리스트 생성 및 관리

### 🌊 백색 소음

- **다양한 소스**: 다양한 내장 백색 소음 소스
- **오디오 효과**: 각 소스별 독립적인 효과 설정
- **스캐터 모드**: 사용자 정의 간격으로 무작위 재생
- **볼륨 제어**: 0-300% 게인 범위

### 🎨 개인화

- **사용자 정의 테마**: HSV 색상 사용자 정의
- **타이머**: 자동 정지 타이머 (최대 23시간 59분)
- **설정 저장**: 효과 매개변수 자동 저장
- **로그 기록**: 문제 해결을 위한 선택적 로그

### 🎛️ 믹서 패널

- **효과 강도**: 모든 효과의 0-100% 조정
- **실시간 미리보기**: 즉각적인 효과 미리보기
- **빠른 리셋**: 원클릭 매개변수 리셋

## 📥 다운로드

### 요구 사항

- Android 10.0 (API 29) 이상
- 최소 100MB 여유 저장 공간

### 설치

[Releases](https://github.com/byxixiaoshao/AVIV/releases) 페이지에서 최신 APK를 다운로드하세요.

**단계**:
1. APK 파일 다운로드
2. 기기에서 "알 수 없는 소스에서 설치" 활성화
3. APK 파일을 열어 설치
4. 설치 후 앱 실행

## 📖 사용법

### 음악 재생

1. 상단 플레이어 바를 아래로 스와이프하여 플레이어 인터페이스 열기
2. 파일 선택기를 통해 음악 파일 추가
3. 믹서 패널을 사용하여 오디오 효과 조정
4. 플레이리스트 생성 및 관리

### 백색 소음

1. 메인 화면에서 백색 소음 소스 선택
2. 소스 카드를 탭하여 재생 시작
3. 카드를 길게 눌러 효과 설정
4. 스캐터 모드를 사용하여 무작위 재생

### 오디오 효과

- **이퀄라이저**: 믹서 → 이퀄라이저 패널
- **크리에이티브 효과**: 믹서 → 음질 패널
- **공간 리버브**: 믹서 → 리버브 패널
- **사운드 포지셔닝**: 믹서 → 공간 오디오 패널
