## 📱 Acerca de

AVIV es una aplicación Android que combina la reproducción de ruido blanco con un reproductor de música, ofreciendo ricos efectos de audio y configuraciones personalizadas para ayudar a los usuarios a relajarse, mejorar la concentración o la calidad del sueño.

## ✨ Características

### 🎵 Reproductor de música

- **Soporte multi-formato**: MP3, WAV, FLAC, AAC y más
- **Audio espacial 3D**: Posicionamiento de sonido, atenuación de distancia
- **Ecualizador de 12 bandas**: Ecualizador profesional
- **Efectos creativos**:
  - Lo-Fi (Baja fidelidad)
  - 8-bit (Sonido de juego retro)
  - Efecto submarino
  - Señal alienígena
  - Efecto megáfono
  - Distorsión
- **Reverberación espacial**: Tamaño de habitación, tiempo de decaimiento, mezcla húmedo/seco
- **Gestión de listas de reproducción**: Crear y gestionar listas de reproducción

### 🌊 Ruido blanco

- **Múltiples fuentes**: Varias fuentes de ruido blanco integradas
- **Efectos de audio**: Configuración independiente de efectos para cada fuente
- **Modo disperso**: Reproducción aleatoria con intervalos personalizables
- **Control de volumen**: Rango de ganancia 0-300%

### 🎨 Personalización

- **Temas personalizados**: Personalización de color HSV
- **Temporizador**: Temporizador de parada automática (hasta 23h 59m)
- **Persistencia de configuración**: Guardado automático de parámetros de efectos
- **Registro**: Registro opcional para solución de problemas

### 🎛️ Panel de mezclador

- **Intensidad de efectos**: Ajuste 0-100% para todos los efectos
- **Vista previa en tiempo real**: Vista previa instantánea de efectos
- **Restablecimiento rápido**: Restablecimiento de parámetros con un clic

## 📥 Descargar

### Requisitos

- Android 10.0 (API 29) o superior
- Al menos 100 MB de espacio de almacenamiento libre

### Instalación

Descargue el último APK desde la página [Releases](https://github.com/byxixiaoshao/AVIV/releases).

**Pasos**:
1. Descargar archivo APK
2. Habilitar "Instalar desde fuentes desconocidas" en su dispositivo
3. Abrir archivo APK para instalar
4. Iniciar la aplicación después de la instalación

## 📖 Uso

### Reproducción de música

1. Deslice hacia abajo la barra superior del reproductor para abrir la interfaz del reproductor
2. Agregue archivos de música a través del selector de archivos
3. Ajuste los efectos de audio usando el panel del mezclador
4. Cree y gestione listas de reproducción

### Ruido blanco

1. Seleccione una fuente de ruido blanco en la pantalla principal
2. Toque la tarjeta de fuente para comenzar la reproducción
3. Mantenga presionada la tarjeta para configurar efectos
4. Use el modo disperso para reproducción aleatoria

### Efectos de audio

- **Ecualizador**: Mezclador → Panel de ecualizador
- **Efectos creativos**: Mezclador → Panel de calidad de audio
- **Reverberación espacial**: Mezclador → Panel de reverberación
- **Posicionamiento de sonido**: Mezclador → Panel de audio espacial
